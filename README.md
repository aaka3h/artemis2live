## ✨ Features

| Feature | Description |
|---|---|
| 📡 **Live Distance** | Real-time km + miles from Earth, updating every second |
| ⚡ **Live Speed** | Current velocity in km/s, km/h and mph |
| 🌕 **Scroll Trajectory** | Scroll to fly Orion from Earth to the Moon and back |
| 🌈 **Gradient Orbit Path** | Color-coded free-return trajectory with milestone labels |
| 🛰️ **DSN Live Status** | Which NASA ground station is tracking Orion right now |
| 📶 **Signal Delay** | Real-time speed-of-light delay vs ISS and Mars |
| ⏱️ **Splashdown Countdown** | Counting down to Pacific Ocean splashdown |
| 📋 **12 Mission Stages** | Every phase from launch pad to splashdown with technical data |
| 🏆 **Distance Record Tracker** | Watching Orion break Apollo 13's 54-year record |
| 📰 **Mission News Feed** | Latest updates from NASA, Space.com, CNN, CSA |
| 👥 **Crew Section** | All 4 astronauts with their historic firsts |
| 📬 **Mission Updates** | Email subscription for key milestones |
| 🇫🇷 **French Version** | Full French translation — the only French Artemis II tracker |

---

## 🎯 Why This Exists

Most Artemis II trackers are English-only and show basic distance/speed. This tracker was built to give **more data** than anything else out there:

- Signal delay based on actual speed of light calculations
- DSN station tracking (which antenna dish is talking to Orion right now)
- Interactive scroll trajectory with stage-by-stage technical details
- **French version** — Jeremy Hansen is Canadian, French-speaking audience had nothing

---

## 🛠️ Tech Stack

- **Pure HTML/CSS/JavaScript** — zero frameworks, zero dependencies
- **Canvas API** — scroll-based trajectory animation
- **Hosted on GitHub Pages** — free, fast, no backend needed
- **Orbital math** — positions computed from known mission parameters and Bezier interpolation
